# Reading the Bits to Binders PDF in Cursor

## Install PDF Extension

1. Press `Cmd+Shift+X` (macOS) to open Extensions
2. Search for **"vscode-pdf"** by tomoki1207
3. Click **Install**
4. Click the PDF file in the Explorer to open it

## Alternative

Right-click the PDF → **Reveal in Finder/File Explorer** → open with your system PDF reader
